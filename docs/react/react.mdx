# React란

**React는 사용자 인터페이스를 만들기 위한 JavaScript 라이브러리다.**

## 선언형 UI

DOM을 직접 조작하는 아니라, UI를 선언하면 데이터에 따라 UI를 React App이 DOM을 조작하여 갱신한다.

## 컴포넌트 기반

UI는 [컴포넌트](./component.mdx) 단위로 나누어 구성 된다.
