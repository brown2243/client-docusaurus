# 타입 추론

## 타입 검사

- 타입 검사란, 어떤 심벌에 대한 각종 대입/참조/연산이 가능한지 확인하는 과정입니다.
- 구체적으로 어떤 맥락 하에 심벌이 가질 수 있는 타입은 무엇인지도 찾아냅니다.

## 제네릭

- 제네릭이란, 타입의 함수이기도 하지만, 타입 간의 관계 그 자체를 표현한 것이기도 합니다.
- 제네릭의 타입 인자에 extends 키워드를 붙여서, 허용하는 슈퍼타입을 지정할 수 있습니다.
